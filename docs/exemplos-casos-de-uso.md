# Exemplos e Casos de uso

## Fluxos principais do sistema

### 1. Cadastro de usuário

Fluxo esperado:

1. acessar a tela de cadastro;
2. informar nome, e-mail, SIAPE e senha;
3. selecionar um ou mais setores;
4. concluir o cadastro;
5. receber o token de autenticação.

### 2. Login

Fluxo esperado:

1. informar SIAPE e senha;
2. autenticar no sistema;
3. acessar o dashboard principal.

### 3. Recuperação de senha

Fluxo esperado:

1. informar o e-mail;
2. validar o código recebido;
3. cadastrar a nova senha.

### 4. Gestão de planos de risco

Fluxo esperado:

1. acessar a tela de planos;
2. filtrar por setor, categoria ou data;
3. criar, editar ou visualizar um plano;
4. registrar ações de tratamento e monitoramento.
